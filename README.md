# Clase2
Markdown es un lenguaje de marcado ligero que permite dar formato a un texto usando símbolos sencillos de teclado. Es el estándar de oro para documentación en GitHub, notas personales y, por supuesto, para hablar conmigo.

Aquí tienes lo esencial para dominarlo en dos minutos:

---

### 1. Títulos

Se usan almohadillas o hashtags (`#`). A más hashtags, más pequeño es el título.

* `# Título Principal`
* `## Subtítulo`
* `### Sección`

### 2. Énfasis (Formato de texto)

* **Negrita**: Encierra el texto entre dos asteriscos: `**texto**`.
* *Cursiva*: Encierra el texto entre un asterisco o guion bajo: `*texto*` o `_texto_`.
* ~~Tachado~~: Usa dos virgulillas: `~~texto~~`.

### 3. Listas

* **Desordenadas**: *Usa asteriscos, más o guiones* (`*`, `+`, `-`).
* **Ordenadas**: *Simplemente usa números seguidos de un punto* (`1.`, `2.`).
* **Listas de tareas**:
* [x] **Tarea terminada**.
* [ ] **Tarea pendiente**.



### 4. Enlaces e Imágenes

* **Enlace**: `[Nombre del sitio](https://url.com)`
* **Imagen**: *Es igual, pero con un signo de exclamación al principio*: `![Texto alternativo](url_de_la_imagen.jpg)`

### 5. Código y Citas

* **Código en línea**: *Usa comillas simples invertidas para resaltar* `palabras técnicas`.
* **Bloque de código**: *Usa tres comillas invertidas (backticks) al principio y al final*.
* **Citas**: *Usa el signo de mayor que (`>`) para crear bloques de texto resaltados*.

### 6. Tablas

*Se construyen con tuberías (`|`) y guiones (`-`)*:

| Herramienta | Facilidad |
| --- | --- |
| Markdown | Alta |
| HTML | Media |
